Introduction to Renting movies system

This project shows you how Renting movies system is working. It contains such information as: customer, movie and rent data.


Methodology

Technologies that were used in this project:

    HTML
    XML
    XML Schema
    XSL
    XSLT

Work content

This project consists files with such type:

    HTML
    XML
    XSD
    XSLT

Elements of the XML

Database is the root element, it contains 5 customer records as a child elements.

Each entry of customer has:

Customer data

  • First name
  • Surname
  • Age
  • Status
  • Job
  • etc.

Account data

  • Login
  • Password
  • Newsletter status
  • etc.

Address

  • Country
  • City
  • District
  • Street
  • Number
  • etc.

Movie rented

  • Title
  • Producer
  • Genre
  • Year
  • Country
  • etc.

HTML table Transformation

XSLT has been used for this transformation of XML to an HTML table with customers.

All entries choosed by random from database. Customers, who didn't bring back movie by the time are marked with red color, others with green.
